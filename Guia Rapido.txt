Comandos do Enviar_Arq:
    !sair -> para sair da conexão
    em breve -> !enviar

Comandos do Receber_Arq:
    !sair -> para sair da conexão
    !receber -> para pedir ao cliente mandar o arquivo

Se você abrir pela primeira vez o programa, tanto o Enviar_Arq e Receber_Arq, você precisara fazer um usuario
o usuario é só o nickname, o nickname é para o host e o cliente serem reconhecido um pelo outro.

(Dica: é bom colocar arquivo com nome pequeno)
